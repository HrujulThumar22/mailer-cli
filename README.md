# mailer-cli

A command line interface to send mail without any hassle.

# Installation

Manually Install mailer from this package

```
$ git clone https://github.com/HrujulThumar22/mailer-cli.git

$ cd mailer-cli/

$npm install -g
```

Check Installtiong by running

```
$ mailer --version
```

# Usage

To use this

```
$ mailer --help
```

---

To initialise:

```
$ mailer -i
```
